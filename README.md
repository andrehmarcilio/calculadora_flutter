# calculadora_legal

Neste projeto desenvolvi uma calculadora usando o framework Flutter.

![20220822_093521.gif](https://user-images.githubusercontent.com/103319187/185936558-c2640d29-74a8-46c0-90af-ba1de01ff289.gif)

## Tecnologias utilizadas

- Gerência de estado com o Bloc
- Widgets: GridView e outros componentes básicos de layout
- Classe 'Calculadora' desenvolvida sem a utilização de plugins externos
